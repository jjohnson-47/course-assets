üìç Distance between Two Points

The fundamental distance formula in three-dimensional space.

Point-to-Point Distance
Given points $A(x_1,y_1,z_1)$ and $B(x_2,y_2,z_2)$: $$d = \sqrt{(x_2 - x_1)^2+(y_2 - y_1)^2+(z_2 - z_1)^2}$$
This is the 3D extension of the Pythagorean theorem and forms the foundation for all other distance calculations.

üìè Distance from Point to Line

Uses the cross product to find the perpendicular distance from a point to a line in 3D space.

Point-to-Line Distance Formula
Given point $P(x_1,y_1,z_1)$ and line $L: \vec{r}(t) = \langle x_0, y_0, z_0 \rangle + t \langle a, b, c \rangle$: $$d = \frac{||\overrightarrow{QP} \times \langle a, b, c \rangle||}{||\langle a, b, c \rangle||} = \frac{||\langle x_1 - x_0, y_1 - y_0, z_1 - z_0 \rangle \times \langle a, b, c \rangle||}{\sqrt{a^2 + b^2 + c^2}}$$

Step-by-Step Process

1 Identify components: Point $P$, point $Q$ on the line, and direction vector $\vec{d} = \langle a, b, c \rangle$
2 Form vector: $\overrightarrow{QP} = \langle x_1 - x_0, y_1 - y_0, z_1 - z_0 \rangle$
3 Cross product: Calculate $\overrightarrow{QP} \times \vec{d}$
4 Apply formula: $d = \frac{||\overrightarrow{QP} \times \vec{d}||}{||\vec{d}||}$
The cross product gives a vector perpendicular to both $\overrightarrow{QP}$ and the line direction. Its magnitude represents the "height" of the parallelogram formed by these vectors.

üìê Distance between Two Lines

Lines in 3D space can have three different relationships, each requiring a different approach.

Case 1: Intersecting Lines

Distance = 0

Lines cross at a point

Case 2: Parallel Lines

Use point-to-line distance

Lines never meet, same direction

Case 3: Skew Lines

Use cross product method

Lines don't intersect and aren't parallel

Checking for Intersection

For lines $L_1: \vec{r}_1(t) = \langle x_1, y_1, z_1 \rangle + t \langle a_1, b_1, c_1 \rangle$ and $L_2: \vec{r}_2(s) = \langle x_2, y_2, z_2 \rangle + s \langle a_2, b_2, c_2 \rangle$: $$\begin{bmatrix} a_1 & -a_2 \\ b_1 & -b_2 \\ c_1 & -c_2 \end{bmatrix} \begin{bmatrix} t \\ s \end{bmatrix} = \begin{bmatrix} x_2 - x_1 \\ y_2 - y_1 \\ z_2 - z_1 \end{bmatrix}$$

Distance between Skew Lines

Skew Lines Distance
$$d = \frac{|(\vec{P_1} - \vec{P_2}) \cdot (\vec{v_1} \times \vec{v_2})|}{||\vec{v_1} \times \vec{v_2}||}$$

Method for Skew Lines

1 Find vector $\vec{n} = \vec{v}_1 \times \vec{v}_2$ orthogonal to both direction vectors
2 Scale to unit vector: $\vec{u} = \frac{\vec{n}}{||\vec{n}||}$
3 Distance = projection of $\overrightarrow{P_1P_2}$ onto $\vec{u}$

üî≤ Distance from Point to Plane

Uses the dot product and normal vector to find perpendicular distance.

Point-to-Plane Distance
Given point $P(x_1, y_1, z_1)$ and plane $ax + by + cz + d = 0$: $$d = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}}$$
Alternative Form with Point on Plane
If plane is $a(x - x_0) + b(y - y_0) + c(z - z_0) = 0$: $$d = \frac{|\vec{n} \cdot \overrightarrow{QP}|}{||\vec{n}||} = \frac{|a(x_1 - x_0) + b(y_1 - y_0) + c(z_1 - z_0)|}{\sqrt{a^2 + b^2 + c^2}}$$
The normal vector $\vec{n} = \langle a, b, c \rangle$ is perpendicular to the plane. The distance is the magnitude of the projection of any vector from the plane to the point onto this normal vector.

üìèüî≤ Distance between Line and Plane

The relationship between a line and plane determines the distance calculation method.

Line Intersects Plane

Distance = 0

Check by substituting line equation into plane equation

Line Parallel to Plane

Distance = point-to-plane distance

Use any point on the line

How to Check Intersection

1 Substitute line equations $x = x_1 + ta_1$, $y = y_1 + tb_1$, $z = z_1 + tc_1$ into plane equation
2 If you get a valid solution for $t$, the line intersects the plane
3 If you get $0 = \text{constant}$ (contradiction), the line is parallel to the plane

üî≤üî≤ Distance between Two Planes

Plane relationships determine the distance calculation approach.

Planes Intersect

Distance = 0

Normal vectors not parallel

Same Plane

Distance = 0

Identical equations

Parallel Distinct Planes

Use point-to-plane formula

Normal vectors parallel, different d values

Distance between Parallel Planes
For planes $a_1x + b_1y + c_1z + d_1 = 0$ and $a_2x + b_2y + c_2z + d_2 = 0$ where $\langle a_1, b_1, c_1 \rangle = k\langle a_2, b_2, c_2 \rangle$: $$d = \frac{|d_1 - d_2|}{||\vec{n}||} = \frac{|a_1x_2 + b_1y_2 + c_1z_2 + d_1|}{\sqrt{a_1^2 + b_1^2 + c_1^2}}$$
Check if normal vectors are parallel by seeing if $\vec{n}_1 = k\vec{n}_2$ for some scalar $k$. If so, the planes are either identical or parallel.

üîë Key Concepts Summary

‚äó
Cross Product Usage

Used for point-to-line and skew line distances. Gives perpendicular vector and area information.

‚Ä¢
Dot Product Usage

Used for point-to-plane distances. Gives projection and perpendicular distance information.

üîç
Check Special Cases

Always check for intersection, parallelism, or identical objects before applying distance formulas.

üìê
Unit Vectors

Normalize vectors (divide by magnitude) to simplify projection calculations and get unit distances.

üìö Study Strategy

Master the point-to-point formula first, then understand how cross products extend this to lines and dot products extend this to planes. Practice identifying the geometric relationship (intersecting, parallel, skew) before applying formulas!
